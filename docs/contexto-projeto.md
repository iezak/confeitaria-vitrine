# Contexto do Projeto

## Tipo de projeto
Projeto extensionista de ADS e também prestação de serviço real.

## Ideia central
Desenvolver uma vitrine virtual para uma confeiteira, com foco em apresentação dos produtos e encaminhamento do cliente para o WhatsApp.

## Escopo fechado da versão final
- Home
- Sobre
- Contato
- catálogo por categorias
- carrossel de produtos por categoria
- ao clicar no produto, abrir o descritivo logo abaixo
- botão de WhatsApp com mensagem pronta
- área admin com login
- CRUD de categorias
- CRUD de produtos
- upload de imagens
- persistência real dos dados

## Fluxo do cliente
1. Cliente entra no site.
2. Visualiza a página inicial.
3. Navega pelas categorias.
4. Vê os produtos em carrossel.
5. Clica em um produto.
6. Vê os detalhes completos abaixo.
7. Clica em “Pedir no WhatsApp”.
8. É encaminhado ao WhatsApp com mensagem pronta.

## Fluxo da administradora
1. Faz login.
2. Acessa o painel admin.
3. Cadastra categorias.
4. Cadastra produtos.
5. Edita ou exclui produtos.
6. Faz upload de imagens.
7. Atualiza o catálogo sem mexer no código.

## O que NÃO entra no escopo
- carrinho
- pagamento online
- área do cliente
- acompanhamento de pedido
- controle de estoque
- cupom de desconto
- cálculo de frete
