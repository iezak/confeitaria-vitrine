# Roadmap de Implementação

## Fase 1 - Estrutura e planejamento
- criar projeto React
- configurar rotas principais
- criar layout base com Header e Footer
- organizar dados iniciais de categorias e produtos

## Fase 2 - Área pública
- criar Home
- criar Sobre
- criar Contato
- ajustar responsividade inicial

## Fase 3 - Catálogo interativo
- criar ProductCard
- criar CategorySection
- criar ProductCarousel
- criar ProductDetails
- criar botão de WhatsApp com mensagem dinâmica

## Fase 4 - Área administrativa
- criar tela de login
- proteger rotas administrativas
- criar dashboard
- implementar CRUD de categorias
- implementar CRUD de produtos

## Fase 5 - Persistência e entrega
- implementar persistência real
- implementar upload de imagens
- executar testes finais
- publicar o sistema
- registrar evidências para a faculdade
